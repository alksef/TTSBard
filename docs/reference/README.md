# Техническая справка

Поддерживаемая справочная информация о конфигурации, событиях и публичных
контрактах проекта. Ручные карты файлов и модулей сохраняются здесь только если
они дают ценность сверх навигации по исходному коду и регулярно обновляются.

Сейчас отдельные ручные каталоги Rust-модулей и Vue-компонентов не ведутся:
они быстро устаревали и дублировали исходное дерево. Добавлять новый справочник
сюда следует только при наличии стабильного публичного контракта.
