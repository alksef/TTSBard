# Stage 26: Внешний вид главного окна в компактном режиме

## Цель

Разделить управление цветом и прозрачностью главного окна и добавить отдельный сценарий, при котором прозрачность используется только в компактном режиме.

## Поведение настроек

В разделе «Настройки → Интерфейс → Главное окно» используются независимые переключатели:

- **Использовать свой цвет** — включает заданный цвет главного окна. Если выключено, используется цвет активной темы.
- **Использовать свою прозрачность** — включает значение прозрачности со шкалы. Если выключено, главное окно остаётся полностью непрозрачным.
- **Применять прозрачность только в компактном режиме** — доступно только при включённой собственной прозрачности.

Алгоритм эффективной прозрачности:

| Своя прозрачность | Только компактный режим | Обычный режим | Компактный режим |
|---|---|---:|---:|
| выкл. | любое | 100% | 100% |
| вкл. | выкл. | значение шкалы | значение шкалы |
| вкл. | вкл. | 100% | значение шкалы |

При переходе в компактный режим значение применяется сразу после изменения состояния. При разворачивании главное окно возвращается к 100%, если включён режим «только в компактном режиме».

## Наследование оформления панелями

Звуковая панель и управление воспроизведением, если выбрано «Как у главного окна», получают сохранённый внешний вид главного окна:

- цвет учитывает переключатель собственного цвета;
- прозрачность учитывает переключатель собственной прозрачности;
- временное состояние компактного режима главного окна на них не распространяется.

Это предотвращает изменение прозрачности отдельных панелей при каждом сворачивании и разворачивании главного окна.

## Кнопка компактного режима

Существующая кнопка с иконками `Minimize2`/`Maximize2` перенесена в нижний правый угол и оформлена треугольником. Логика переключения, размеры окна, блокировка на время анимации, `title` и доступные подписи сохранены.

## Реализация

- `src-tauri/src/config/windows.rs` — сохранение двух новых флагов главного окна и эффективная прозрачность для наследуемого оформления.
- `src-tauri/src/commands/window.rs`, `src-tauri/src/lib.rs` — команды изменения флагов.
- `src/types/settings.ts` — DTO настроек.
- `src/components/settings/SettingsInterface.vue` — элементы управления.
- `src/App.vue` — вычисление прозрачности с учётом компактного режима.
- `src/components/MinimalModeButton.vue` — треугольная кнопка.

Миграция старых значений прозрачности не выполняется: новая настройка собственной прозрачности по умолчанию выключена.
