# Пользовательская документация

Руководства по установке, настройке и использованию TTSBard. Документы должны
описывать наблюдаемое поведение актуальной версии приложения, а не внутренний
процесс реализации.

## Документы

- [Руководство пользователя](./guide.md) — возможности, настройка и решение
  типовых проблем.
- [AI-функции редактора](./ai-text.md) — коррекция, грамматика, продолжение
  текста и настройка AI-провайдеров.
- [Аудиоэффекты](./audio-effects.md) — доступная постобработка и актуальный
  аудиопайплайн.
- [Настройка DSP](./dsp-settings.md) — практические рекомендации по EQ,
  компрессору и лимитеру.
- [Обзор возможностей для стриминга](./presentation.md) — краткая презентация
  основных пользовательских сценариев.
