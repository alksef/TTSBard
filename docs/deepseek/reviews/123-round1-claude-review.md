## Что реализовал DeepSeek

### 1. Backend (Rust)
- **`tauri.conf.json`**: Отключены системные рамки (`"decorations": false`) и включена прозрачность (`"transparent": true`) для главного окна (`main`).
- **`capabilities/default.json`**: Добавлены разрешения на управление окном из фронтенда (`allow-minimize`, `allow-close`).
- **`config/windows.rs`**:
  - `WindowPosition` переименован в `MainWindowSettings` с добавлением полей `custom_background`, `opacity` и `bg_color`.
  - В `SoundPanelWindowSettings` и `PlaybackWindowSettings` добавлено поле `appearance_source: String` (по умолчанию `"own"` при миграции, `"main"` для новых установок).
  - Настроена обратная совместимость при миграции старого `windows.json`.
  - Реализованы getters/setters в `WindowsManager`.
- **`commands/window.rs`**:
  - Добавлен вспомогательный метод `resolve_main_appearance()` для получения текущих эффективных параметров оформления главного окна (с учётом выбранной темы при выключенном пользовательском цвете).
  - Добавлены 8 новых Tauri-команд для получения/изменения параметров оформления и источника наследования.
  - Любые изменения инициируют отправку событий `settings-changed`, `soundpanel-appearance-update` и `playback-appearance-update` для мгновенного обновления окон.
- **`commands/playback_window.rs` & `soundpanel/bindings.rs`**:
  - Команды `pc_get_appearance` и `sp_get_floating_appearance` адаптированы для поддержки наследования стиля главного окна, если `appearance_source == "main"`.
- **`lib.rs`**: Новые команды зарегистрированы в `tauri::generate_handler!`.

### 2. Frontend (Vue / TS)
- **`types/settings.ts`**: Обновлены TypeScript интерфейсы настроек окон.
- **`base.css`**: Фоны `html` и `body` сброшены в `transparent !important`.
- **`App.vue`**:
  - Внедрён кастомный Title Bar (`app-titlebar`) с кнопками Свернуть (иконка Lucide `Minus`) и Закрыть (иконка Lucide `X`), вызывающими соответствующие API окна.
  - Реализован computed-стиль `appStyle` для `.app-container`, вычисляющий фоновый цвет с альфа-каналом и накладывающий его поверх оригинальных градиентов приложения.
  - Изменён лейаут контейнеров для поддержки вертикальной шапки и горизонтального расположения сайдбара и контента.
- **`SettingsPanel.vue` & `SettingsInterface.vue`**:
  - Добавлена новая вкладка настроек «Интерфейс» с иконкой Palette.
  - Перенесён селектор Темы из вкладки «Общие».
  - Выведены независимые настройки цвета и прозрачности для главного окна, звуковой панели и панели воспроизведения, с поддержкой селектора источника стиля.
- **`SettingsGeneral.vue`**: Удалён блок управления темой.
- **`SoundPanelApp.vue` & `PlaybackTab.vue`**:
  - Локальные настройки цвета/прозрачности удалены из звуковой панели и вкладки управления воспроизведением. Во вкладку воспроизведения добавлена локализованная заглушка о переносе настроек в раздел «Интерфейс».
