## Шаг 0: tokio-util

```bash
grep "tokio-util" src-tauri/Cargo.toml
```
Если нет — добавить в `[dependencies]`: `tokio-util = { version = "0.7", features = ["sync"] }`

## Шаг 1: `state.rs` — добавить поле shutdown

```rust
use tokio_util::sync::CancellationToken;

pub struct AppState {
    // ... все существующие поля ...
    pub shutdown: CancellationToken,
}
// В AppState::new(): shutdown: CancellationToken::new(),
```

## Шаг 2: `setup.rs` — передать clone() token серверам

В `init_webview_server` и `init_twitch_client` добавить:
```rust
let shutdown = app_state.shutdown.clone();
app_state.runtime.spawn(async move {
    crate::servers::run_webview_server(..., shutdown).await;
});
```

## Шаг 3: `servers/webview.rs` — параметр + select!

Добавить `shutdown: CancellationToken` в сигнатуру `run_webview_server`.
В основном loop заменить timeout+recv на select!:

```rust
tokio::select! {
    biased;
    _ = shutdown.cancelled() => {
        info!("[WEBVIEW] ⛔ Shutdown signal");
        server.stop();
        server_handle.abort();
        return;
    }
    result = tokio::time::timeout(
        tokio::time::Duration::from_secs(1),
        webview_rx.recv(),
    ) => {
        match result {
            Ok(Some(event)) => { /* существующая обработка event */ }
            Ok(None) => { return; }
            Err(_) => {}
        }
    }
}
```
В disabled loop — аналогичный select! с `shutdown.cancelled()`.

## Шаг 4: `servers/twitch.rs` — параметр + select!

Прочитай `run_twitch_client`. Найди main loop/ожидание событий.
Добавить `shutdown: CancellationToken` параметр и:
```rust
tokio::select! {
    biased;
    _ = shutdown.cancelled() => { info!("[TWITCH] ⛔ Shutdown"); return; }
    event = twitch_rx.recv() => { /* обработка */ }
}
```

## Шаг 5: `commands/mod.rs` — `quit_app`

```rust
if let Some(state) = app_handle.try_state::<AppState>() {
    state.shutdown.cancel();
    info!("Shutdown token cancelled — all servers notified");
    std::thread::sleep(std::time::Duration::from_millis(600));
}
```
Убрать или оставить как fallback блок `webview_event_sender.send(AppEvent::Quit)`.
