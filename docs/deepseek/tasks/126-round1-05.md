# Task 126-round1-05: Compact editor settings section spacing

В `src/components/settings/SettingsEditor.vue` уменьши большой вертикальный промежуток
между карточками «Быстрый редактор» и «Проверка орфографии».

Причина: `.settings-editor` уже использует `gap: 1.5rem`, а `.settings-section` повторно
добавляет `margin-bottom: 1.5rem`, поэтому расстояние складывается. Убери дублирование:
оставь один компактный промежуток через `gap` (например `0.75rem`) и обнули `margin-bottom`
у секций, сохранив `padding`, border и последний layout. Не меняй template и поведение.

Проверь `npx vue-tsc --noEmit` и `git diff --check`.
