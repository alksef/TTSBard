# Task 121-round1-01: Custom OpenAI-compatible AI Provider

План доработки: `docs/stage/19-custom-ai-provider.md` (читать обязательно).

## Описание задачи
Необходимо реализовать кастомный ИИ провайдер (совместимый с OpenAI API), позволяющий настраивать API URL (base URL), API-ключ/токен, имя модели и использование прокси.

Выполни доработку строго по следующим шагам.

---

## Шаг 1. Конфигурация в Rust (`src-tauri/src/config/`)

1. **`src-tauri/src/config/settings.rs`**:
   - Добавь `Custom` в перечисление `AiProviderType`:
     ```rust

         #[default]
         OpenAi,
         ZAi,
         DeepSeek,
         Custom,
     }
     ```
   - Создай структуру настроек `AiCustomSettings`:
     ```rust
     /// Custom OpenAI-compatible settings for AI text correction
     #[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
     pub struct AiCustomSettings {
         pub url: Option<String>,
         pub api_key: Option<String>,
         #[serde(default)]
         pub use_proxy: bool,
         #[serde(default = "default_custom_model")]
         pub model: String,
     }

     fn default_custom_model() -> String {
         "deepseek-chat".to_string()
     }

     impl Default for AiCustomSettings {
         fn default() -> Self {
             Self {
                 url: None,
                 api_key: None,
                 use_proxy: false,
                 model: default_custom_model(),
             }
         }
     }
     ```

   - Обнови реализацию `Default` для `AiSettings` (инициализируй `custom: AiCustomSettings::default()`).
   - В реализации `SettingsManager` добавь методы:

     pub fn set_ai_custom_use_proxy(&self, enabled: bool) -> Result<()> {

     }

     pub fn get_ai_custom_model(&self) -> String {

     }

         self.update_field("/ai/custom/model", &model)
     }

## Шаг 2. AI Клиент в Rust (`src-tauri/src/ai/`)

1. **Создай новый файл `src-tauri/src/ai/custom.rs`**:

   - Если `settings.custom.url` равен `None` или пуст, возвращай ошибку `AiError::NotConfigured("Custom API URL not set".to_string())`.
   - Если `settings.custom.api_key` равен `None` или пуст, возвращай ошибку `AiError::NotConfigured("Custom API key not set".to_string())`.
   - Используй `with_api_base` при конструировании `OpenAIConfig`.
   - Поддержи прокси на основе `settings.custom.use_proxy`.

2. **`src-tauri/src/ai/mod.rs`**:
   - Подключи модуль: `pub mod custom;`
   - Добавь вариант `Custom(custom::CustomClient)` в `pub enum AiProvider`.
   - Обнови `AiProvider::correct()` для поддержки `AiProvider::Custom(client)`.
   - Обнови `create_ai_client()`:

## Шаг 3. Tauri Команды (`src-tauri/src/commands/ai.rs` & `src-tauri/src/lib.rs`)

1. **`src-tauri/src/commands/ai.rs`**:
   - Реализуй команды:
     ```rust
     #[tauri::command]
     pub fn set_ai_custom_url(
         settings_manager: State<'_, SettingsManager>,
         state: State<'_, AppState>,
         url: String,
     ) -> Result<(), String> {
         if url.trim().is_empty() {
             return Err("API URL cannot be empty".into());
         }
         settings_manager.set_ai_custom_url(Some(url))
             .map_err(|e| format!("Failed to save Custom API URL: {}", e))?;
         state.invalidate_ai_client();
         Ok(())
     }

     #[tauri::command]
     pub fn set_ai_custom_api_key(
         settings_manager: State<'_, SettingsManager>,
         state: State<'_, AppState>,
         key: String,
     ) -> Result<(), String> {
         if key.trim().is_empty() {
             return Err("API key cannot be empty".into());
         }
         settings_manager.set_ai_custom_api_key(Some(key))
             .map_err(|e| format!("Failed to save Custom API key: {}", e))?;
         state.invalidate_ai_client();
         Ok(())
     }

     #[tauri::command]
     pub fn set_ai_custom_model(
         settings_manager: State<'_, SettingsManager>,
         model: String,
     ) -> Result<(), String> {
         if model.trim().is_empty() {
             return Err("Model cannot be empty".into());
         }
         settings_manager.set_ai_custom_model(model)
             .map_err(|e| format!("Failed to save Custom model: {}", e))
     }

     #[tauri::command]
     pub fn set_ai_custom_use_proxy(
         settings_manager: State<'_, SettingsManager>,
         state: State<'_, AppState>,
         enabled: bool,
     ) -> Result<(), String> {
         settings_manager.set_ai_custom_use_proxy(enabled)
             .map_err(|e| format!("Failed to save Custom proxy setting: {}", e))?;
         state.invalidate_ai_client();
         Ok(())
     }

     #[tauri::command]
     pub fn get_ai_custom_model(
         settings_manager: State<'_, SettingsManager>,
     ) -> String {
         settings_manager.get_ai_custom_model()
     }
     ```

2. **`src-tauri/src/lib.rs`**:
   - Добавь импорты новых команд из `commands::ai`.
   - Зарегистрируй новые команды в `tauri::Builder::default().invoke_handler(...)`.
