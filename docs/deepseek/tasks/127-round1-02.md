# Task 127-round1-02: UI and InputPanel quick editor modes

Реализуй frontend-часть по плану `docs/deepseek/plan/127-2026-07-17-quick-editor-behavior-mode.md`.
Backend DTO/command уже переведены на режимы задачей `127-round1-01`.

## Файлы

- `src/types/settings.ts`
- `src/components/settings/SettingsEditor.vue`
- `src/components/InputPanel.vue`

## Требования

1. `EditorSettingsDto.quick` заменить на union `'disabled' | 'collapse' | 'return_focus'`.
2. В SettingsEditor сделать отдельную карточку «Быстрый редактор», мелкое описание
   «Реакция на Enter, Esc и отправку текста в TTS», три radio-варианта и сохранение через
   `set_editor_quick` с новым строковым значением.
3. Исправить layout `.settings-editor .setting-row`, сохранив вертикальное размещение
   label/hint.
4. В InputPanel заменить boolean-проверки на режим:
   - disabled: не скрывать/не возвращать focus;
   - collapse: оставить текущий hide;
   - return_focus: вызвать `invoke('return_to_previous_window')`, не скрывать main.
   Для Enter возврат выполнять после отправки текста в TTS; для Esc — сразу.
5. Сохранить текущую очистку текста и обработку ошибок. Не менять spellcheck.

После изменений запусти `npx vue-tsc --noEmit` и `git diff --check`.
