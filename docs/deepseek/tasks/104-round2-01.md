# Task 104-round2-01: История фраз — две кнопки (replace/append), дефолт = append

План: `docs/deepseek/plan/104-2026-07-08-history-select-append-and-undo.md`, **Часть 1 (UX)**.
Часть 2 (undo) — уже сделана в round1, НЕ трогать.

## Решение пользователя
- У каждой записи истории — **две микрокнопки**: «вставить с заменой» и «добавить в конец».
- Дефолт-клик по записи (без кнопок) = **append** (добавить в конец).
- Confirm при replace: **оставить** (защита от misclick).

## Текущее поведение
`src/components/InputPanel.vue:254-260`:
```ts
function selectPhrase(newText: string) {
  const currentText = text.value
  if (currentText.trim() && currentText !== newText) {
    if (!confirm('Заменить текущий текст на выбранную фразу?')) return
  }
  text.value = newText
}
```

## Фикс

### `src/components/InputPanel.vue`
1. Заменить `selectPhrase` на две функции + дефолт = append:
   ```ts
   function appendPhrase(newText: string) {
     const currentText = text.value
     if (!currentText.trim()) {
       text.value = newText
       return
     }
     // добавить в конец с разделителем-пробелом (не дублировать если уже endsWith space)
     const sep = currentText.endsWith(' ') ? '' : ' '
     text.value = currentText + sep + newText
   }

   function replacePhrase(newText: string) {
     const currentText = text.value
     if (currentText.trim() && currentText !== newText) {
       if (!confirm('Заменить текущий текст на выбранную фразу?')) return
     }
     text.value = newText
   }
   ```

   для дефолт-клика). Изменить биндинги:

## Поведение (runtime)
- Клик по записи → текст добавляется в конец (с пробелом-разделителем). Если поле пусто — просто
  вставляется фраза.
- Кнопка «Добавить» → то же.
- Кнопка «Заменить» → confirm (если поле не пусто) → замена.
- Undo (Ctrl+Z) работает после всех трёх (через round1-фикс).
