# Task 107-round1-01: Переключение орфографии падает «missing required key value»

План: `docs/deepseek/plan/107-2026-07-09-spellcheck-toggle-wrong-arg-key.md` (читать обязательно).

## Симптом (от пользователя)
Включение проверки орфографии (offline) даёт ошибку:
`Ошибка переключения орфографии: invalid args \`value\` for command
\`set_editor_spellcheck_enabled\`: command set_editor_spellcheck_enabled missing required key value`

## Корневая причина (найдена Claude)
Бэкенд ждёт параметр `value` (`src-tauri/src/commands/mod.rs:1059`,
`pub fn set_editor_spellcheck_enabled(value: bool, ...)`).
Фронт шлёт ключ `enabled`:
`src/components/settings/SettingsEditor.vue:21`
`await invoke('set_editor_spellcheck_enabled', { enabled: newValue })`
→ Tauri: «missing required key value».

Доказательство — соседний тоггл в том же файле (строка 32) зовёт правильно:
`await invoke('set_editor_quick', { value: newValue });` ✅

## Задача DeepSeek

### Фикс (точечный)
Файл `src/components/settings/SettingsEditor.vue`, строка 21:
```ts
// было:
await invoke('set_editor_spellcheck_enabled', { enabled: newValue })
// стало:
await invoke('set_editor_spellcheck_enabled', { value: newValue })
```

### Проверить родственные команды (сверить с сигнатурами бэкенда)
В `SettingsEditor.vue` и связных компонентах поискать другие `invoke(...)` к spellcheck-командам:
- `set_editor_spellcheck_source` (бэкенд `commands/mod.rs:1082`, параметр `value: SpellSource`)
  — если зовётся, ключ должен быть `value`, не `source`. Поправить если рассинхрон.
- `get_editor_spellcheck_enabled` / `get_editor_spellcheck_source` — без аргументов, не трогать.

Если найден рассинхрон — поправить по тому же принципу (сверить с `#[tauri::command]` сигнатурой).

## Верификация
1. `npx vue-tsc --noEmit` — 0/0.
2. В отчёте: какие строки изменены, найдены ли ещё рассинхроны аргументов spellcheck-команд.

## Главное правило
НЕ трогать онлайн-режим орфографии (`check_spelling_online`) — это отдельный баг (BUG-02,
`docs/bugs/02-spellcheck-online-source-calls-missing-command.md`). НЕ переименовывать параметр
на бэкенде — править фронт под контракт `value`.

## Runtime (пользователь проверит)
- Настройки редактора → тоггл орфографии → НЕ ошибка, «Настройка сохранена».
- Включить → слова с ошибками подчёркиваются (offline). Выключить → перестают.
- Стабильно при повторных переключениях.
