# Task 126-round1-01: backend настройки и возврат foreground

Реализуй backend-часть по плану `docs/deepseek/plan/126-2026-07-17-main-window-return-focus-hotkey.md`.

## Область файлов

- `src-tauri/src/config/hotkeys.rs`
- `src-tauri/src/config/dto.rs`
- `src-tauri/src/config/settings.rs`
- `src-tauri/src/state.rs`
- `src-tauri/src/hotkeys.rs`
- `src-tauri/src/commands/window.rs`
- `src-tauri/src/lib.rs`
- при необходимости существующий Windows helper в `src-tauri/src/window.rs`

## Требования

1. Добавь `return_previous_window: Hotkey` в `HotkeySettings`, serde default и default
   `Ctrl+F`. Расширь DTO, TypeScript пока не меняй.
2. Расширь `set_hotkey`/`reset_hotkey_to_default` именем `return_previous_window` и
   проверкой конфликтов с каждым существующим global shortcut. Это поле не регистрируй
   в `register_from_settings`.
3. Добавь runtime state для сохранённого внешнего foreground HWND.
4. В `handle_main_window` до любой активации окна получай текущий foreground HWND,
   фильтруй собственные окна TTSBard и сохраняй валидный внешний HWND.
5. Добавь Tauri command `return_to_previous_window`: снять `always_on_top` у main,
   вызвать Win32 `SetForegroundWindow` для сохранённого HWND, не скрывать main. Ошибки
   отсутствующего/невалидного HWND обрабатывай мягко и понятно.
6. Сохрани cfg(windows)/non-windows совместимость и добавь unit-тесты для default,
   serde backward compatibility и conflict validation, где это возможно.
7. Не менять NumPad Control, soundpanel bindings и playback behavior.

После изменений запусти `cargo check` и `git diff --check`.
