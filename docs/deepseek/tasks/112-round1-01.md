## Паттерн миграции

Добавить параметр `settings_manager: State<'_, SettingsManager>`, убрать
`SettingsManager::new()`. Пример:

```rust
// БЫЛО:
#[tauri::command]
pub fn set_speaker_device(device_id: Option<String>) -> Result<(), String> {
    SettingsManager::new()
        .and_then(|mgr| mgr.set_speaker_device(device_id))
        .map_err(|e| e.to_string())
}

// СТАЛО:
#[tauri::command]
pub fn set_speaker_device(
    device_id: Option<String>,
    settings_manager: State<'_, SettingsManager>,
) -> Result<(), String> {
    settings_manager
        .set_speaker_device(device_id)
        .map_err(|e| e.to_string())
}
```
