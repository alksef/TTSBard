# Task 125 round 1 — исправить диагностику и обнаружение libclang

Реализуй план из `docs/deepseek/plan/125-build-debug-libclang.md`.

Работай только с `scripts/build.ps1` и, если действительно нужно, с одним
существующим документом о сборке. Не изменяй `src-tauri/Cargo.toml`, Rust-код,
lock-файлы или локальные пользовательские правки.

Сохрани UTF-8 BOM в `scripts/build.ps1`: скрипт должен корректно читаться
Windows PowerShell 5.1 и PowerShell 7.

После правок проверь синтаксис/логику скрипта доступными локальными средствами.
Не утверждай, что полная Tauri-сборка успешна, если `libclang.dll` реально не
найден.
