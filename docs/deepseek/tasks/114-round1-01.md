# Task 114 / Round 1 / Iteration 01 — убрать настройки окна из раздела SoundPanel

## Контекст

В `src/components/SoundPanelTab.vue` после таблицы привязок находится `<section class="appearance-section">` с цветом, прозрачностью, click-through, `stayVisible` и preview. Эти настройки относятся к floating-окну и должны управляться из общих настроек интерфейса, а в разделе звуковой панели больше не должны дублироваться.

## Что сделать

1. Удали из шаблона `SoundPanelTab.vue` весь блок `.appearance-section`, включая checkbox click-through и checkbox «Оставлять окно видимым». Не трогай диалоги добавления звука/набора и таблицу привязок.
2. Удали из destructuring `useSoundPanel()` в `SoundPanelTab.vue` ставшие ненужными поля и методы.
3. В `src/composables/useSoundPanel.ts` удали только код, используемый исключительно удалённым блоком: `opacity`, `bgColor`, `clickthroughEnabled`, `stayVisible`, `previewStyle`, `hexToRgba`, `loadAppearanceSettings`, `saveOpacity`, `saveBgColor`, `saveClickthrough`, `saveStayVisible`, а также их загрузку/синхронизацию/listener, если после удаления они не нужны другим функциям composable. Сохрани общие `useAppSettings`-зависимости, если они нужны для остального поведения.
4. Не удаляй backend-команды `sp_set_floating_*`, `sp_get_*` и поля настроек: они используются floating-окном и `SettingsInterface.vue`.
5. Удали ставшие неиспользуемыми CSS-правила appearance section только если они больше нигде не нужны в этом компоненте.
