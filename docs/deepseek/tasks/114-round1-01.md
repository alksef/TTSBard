# Task 114-round1-01: Один Tokio runtime вместо RT-2…RT-5

План: `docs/deepseek/plan/114-2026-07-11-single-tokio-runtime.md` (читать обязательно).

## Предусловие

После task 113: `webview_event_sender` — уже `UnboundedSender<AppEvent>`,
`run_webview_server` — уже принимает `UnboundedReceiver<AppEvent>`.

## Контекст

`setup.rs` создаёт 4 лишних Tokio-рантайма в `init_webview_server` (~строки 443-499)
и `init_twitch_client` (~строки 502-554) через паттерн:
`thread::spawn → Builder::new_multi_thread().build() → rt.block_on(...)`.
`AppState` уже имеет `pub runtime: Arc<tokio::runtime::Runtime>` — использовать его.

## Задача

### 1. `setup.rs` — `init_webview_server`: убрать оба thread::spawn+runtime, заменить:

```rust
fn init_webview_server(app_state: &AppState, app_handle: AppHandle) {
    let webview_settings = app_state.webview_settings.clone();
    let (webview_tx, webview_rx) = tokio::sync::mpsc::unbounded_channel::<AppEvent>();
    app_state.set_webview_event_sender(webview_tx);

    app_state.runtime.spawn(async move {
        crate::servers::run_webview_server(webview_settings, app_handle, webview_rx).await;
    });
}
```

### 2. `servers/webview.rs` — autostart inline в начале run_webview_server (до loop):

```rust
{
    let settings = webview_settings.read().await;
    if settings.start_on_boot && !settings.enabled {
        drop(settings);
        webview_settings.write().await.enabled = true;
        info!("[WEBVIEW] Auto-start on boot: enabled");
    }
}
```

### 3. `setup.rs` — `init_twitch_client`: то же самое:

```rust
fn init_twitch_client(app_state: &AppState, app_handle: AppHandle) {
    let app_state_clone = app_state.clone();
    let twitch_rx = app_state.twitch_event_tx.subscribe();

    app_state.runtime.spawn(async move {
        crate::servers::run_twitch_client(app_state_clone, app_handle, twitch_rx).await;
    });
}
```

### 4. `servers/twitch.rs` — autostart inline в начале run_twitch_client (до loop):

Прочитай файл, найди main loop. Добавь в начало:

```rust
{
    let settings = app_state.twitch_settings.read().await;
    if settings.start_on_boot && settings.enabled && settings.is_valid().is_ok() {
        drop(settings);
        info!("[TWITCH] Auto-start on boot");
        app_state.send_twitch_event(crate::events::TwitchEvent::Restart);
    }
}
```
