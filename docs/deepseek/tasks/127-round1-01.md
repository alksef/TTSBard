# Task 127-round1-01: Backend quick editor behavior mode

Реализуй backend-часть по плану `docs/deepseek/plan/127-2026-07-17-quick-editor-behavior-mode.md`.

## Файлы

- `src-tauri/src/config/settings.rs`
- `src-tauri/src/config/dto.rs`
- `src-tauri/src/commands/mod.rs`
- при необходимости `src-tauri/src/lib.rs`

## Требования

1. Замени `EditorSettings.quick: bool` на сериализуемый enum `QuickEditorMode` со
   стабильными lowercase-значениями `disabled`, `collapse`, `return_focus`.
2. Обеспечь backward-compatible deserialization старых `quick: false/true`:
   false → disabled, true → collapse. Новые настройки сохранять строкой.
3. Обнови `EditorSettings::default()` на `disabled`, `EditorSettingsDto` и
   `AppSettingsDto` на новый тип/строку.
4. Замени `set_editor_quick(bool)` на команду/сеттер, принимающий новый режим;
   сохрани имя IPC `set_editor_quick`, если это возможно без неоднозначности.
   `get_editor_quick` должен вернуть новый режим либо добавь совместимый typed getter.
5. Добавь unit-тесты миграции bool/string, default и round-trip JSON.
6. Не меняй поведение InputPanel в этом task и не трогай hotkeys/NumPad Control.

После изменений запусти `cargo check` и `git diff --check`.
