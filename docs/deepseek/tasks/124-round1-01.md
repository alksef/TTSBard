# Task 124 / Round 1 / Iteration 1

Исправь регрессию запуска Windows debug-сборки по плану `docs/deepseek/plan/124-build-debug-encoding.md`.

Контекст:

- Репозиторий: `D:\RustProjects\app-tts-v2`.
- `scripts/build-debug.bat` запускает `powershell -NoProfile -ExecutionPolicy Bypass -File scripts/build.ps1 -Mode debug`.
- Сейчас `scripts/build.ps1` — UTF-8 без BOM. Windows PowerShell 5.1 читает кириллицу/длинное тире как ANSI и выдаёт parser error до сборки.

Требования:

1. Добавь UTF-8 BOM в начало `scripts/build.ps1`.
2. Не меняй функциональный текст скрипта, `.bat`, исходники приложения и чужие незакоммиченные файлы.
3. Проверь первые байты файла и выполни релевантные проверки. Для debug wrapper используй `cmd /c scripts\\build-debug.bat`; если полноценная сборка запускается долго или падает по независимой причине, зафиксируй точную причину и не маскируй её.

Не считай задачу выполненной только по чек-листу: проверь фактический diff и отсутствие parser error.
