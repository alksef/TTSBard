# Task 97-round1-01: A-Z клавиши саундпанели — layout-независимо

План: `docs/deepseek/plan/97-2026-07-08-soundpanel-keys-layout-independent.md` (читать).

## Корень
`src-soundpanel/SoundPanelApp.vue:148` — `const key = e.key.toUpperCase()` зависит от раскладки.
На RU-раскладке физическая A даёт `e.key === "ф"` → не проходит `/^[A-Z]$/` → звук не играет.

## Фикс
В `src-soundpanel/SoundPanelApp.vue` функция `onKeydown` — извлекать букву из **`e.code`**
(`"KeyA".."KeyZ"`), не из `e.key`. Пример:

```ts
function codeToLetter(code: string): string | null {
  if (code.length === 4 && code.startsWith('Key')) {
    const letter = code[3].toUpperCase()
    if (letter >= 'A' && letter <= 'Z') return letter
  }
  return null
}

function onKeydown(e: KeyboardEvent) {
  if (e.key === 'Escape') { closeWindow(); return }
  if (e.ctrlKey || e.shiftKey || e.altKey || e.metaKey) return
  const key = codeToLetter(e.code)
  if (!key) return
  const b = bindings.value.find(x => x.key === key)
  if (b) {
    e.preventDefault()
    invoke('sp_play_binding', { key }).then(() => closeWindow())
  } else {
    showNoBinding(key)
  }
}
```

Текущий `onKeydown` находится в районе строки 141-159. Сохранить Escape, modifier-guards,
preventDefault, showNoBinding — всё как есть, заменить только источник клавиши.
