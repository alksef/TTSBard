## Fix A — `src-tauri/src/rate_limiter.rs`

### Что менять

Функция `with_config` (строки 30–37). Сейчас:
```rust
pub fn with_config(requests_per_minute: u32, burst_size: u32) -> Self {
    let quota = Quota::per_minute(NonZeroU32::new(requests_per_minute).unwrap())
        .allow_burst(NonZeroU32::new(burst_size).unwrap());
    ...
}
```

При аргументе `0` — panic. Исправление: изменить типы параметров на `NonZeroU32`:
```rust
pub fn with_config(requests_per_minute: NonZeroU32, burst_size: NonZeroU32) -> Self {
    let quota = Quota::per_minute(requests_per_minute)
        .allow_burst(burst_size);
    ...
}
```

`NonZeroU32` уже импортирован (`use std::num::NonZeroU32`).

### Тесты
В том же файле (строки 67–90) тесты вызывают `with_config(10, 2)` и `with_config(2, 1)`.
Обновить вызовы:
```rust
TtsRateLimiter::with_config(NonZeroU32::new(10).unwrap(), NonZeroU32::new(2).unwrap())
TtsRateLimiter::with_config(NonZeroU32::new(2).unwrap(), NonZeroU32::new(1).unwrap())
```

### Добавить тест нулевых аргументов
```rust
// Compile-time safety — тест на то что 0u32 не компилируется, не нужен.
// Достаточно существующих тестов.
```
