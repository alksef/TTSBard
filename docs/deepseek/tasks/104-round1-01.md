# Task 104-round1-01: Extract audio devices tab

Прочитай:

- `docs/stage/35-audio-panel-subpanels.md`
- `docs/deepseek/plan/104-audio-panel-subpanels.md`

Реализуй только первый раунд: вынеси из `src/components/AudioPanel.vue` разметку и локально необходимую логику вкладки устройств в новый компонент `src/components/audio/AudioDevicesTab.vue`.

Вкладка должна сохранить текущее поведение динамиков, виртуального микрофона, тестирования устройств, загрузки/обновления списка и отображения ошибок. Используй props/events для связи с родителем либо аккуратно передай существующие данные/обработчики; не меняй backend-команды и composables. Родитель должен остаться композицией вкладок и передать только необходимые зависимости.

Не выноси эффекты, preview или DSP в этой задаче. Не меняй стили визуально: перенеси необходимые scoped-стили без redesign и не создавай дубликаты, если стиль можно оставить в родителе безопасно.

После реализации запусти `npx vue-tsc --noEmit`, проверь diff и сообщи изменённые файлы. Не форматируй весь Rust-проект и не трогай несвязанные пользовательские изменения.
