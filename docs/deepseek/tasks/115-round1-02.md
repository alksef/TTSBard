# Task 115 / Round 1 / Iteration 02 — восстановить секцию главного окна

## Review finding

При добавлении checkbox в `SettingsInterface.vue` предыдущая итерация изменила не тот участок: из секции «Главное окно» исчезли controls прозрачности и `mainOpacityCompactOnly`, а checkbox `Скрывать при потере фокуса` был временно вставлен туда. Из-за этого `vue-tsc` сообщает о неиспользуемых `mainOpacityDisabled`, `mainCompactOnlyDisabled`, `saveMainOpacity` и `toggleMainOpacityCompactOnly`.

## Что сделать

1. В `src/components/settings/SettingsInterface.vue` восстанови в секции «Главное окно» исходные controls без изменений:
   - поле/слайдер прозрачности главного окна с `v-model.number="mainOpacity"`, `:disabled="mainOpacityDisabled"`, `@change="saveMainOpacity"`;
   - checkbox «Применять прозрачность только в компактном режиме» с `:checked="mainOpacityCompactOnly"`, `:disabled="mainCompactOnlyDisabled"`, `@change="toggleMainOpacityCompactOnly"` и исходным hint.
2. Удали ошибочно вставленный `Скрывать при потере фокуса` из секции «Главное окно», если он там остался.
3. Оставь ровно один checkbox `Скрывать при потере фокуса` внутри секции «Звуковая панель», после её блока цвета/прозрачности, с `:checked="spHideOnBlur"` и `@change="toggleSpHideOnBlur"`.
4. Не меняй backend и остальные настройки.
