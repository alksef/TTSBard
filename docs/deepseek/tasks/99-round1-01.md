# Task 99-round1-01: Превью быстрой вставки (\ключ → замена) через autocomplete

План: `docs/deepseek/plan/99-2026-07-08-quick-insert-preview.md` (читать).
Решение пользователя: **Вариант A** — autocomplete-popup (переиспользовать существующий
autocompletion в TtsEditor).

## Цель
Когда пользователь ввёл `\ключ` (или `%key`) и ещё не нажал пробел — показать popup с matching
ключами из replacements/usernames и **preview результата замены** (detail «→ Александр»), чтобы
видеть на что будет заменено.

## Контекст (читать обязательно)
- `src/components/editor/TtsEditor.vue`:
  - `createKeymap()` (~строка 240-284) — обработчик пробела: парсит `\\([^\s]+)$` (replacement)
    и `%([^\s]+)$` (username) перед курсором, подставляет по пробелу. **Не ломать.**
  - `autocompletion({ override: [hybridSource], ... })` (~строка 296) — текущий источник
    (фразы, `useTextCompletion.ts`).
  - props: `rep` (replacements Record) и `usr` (usernames Record) — см. `replacementsRecord`/
    `usernamesRecord` в InputPanel, и `createKeymap` где они используются (`rep.value[key]`).
- Подстановка по пробелу — ОСТАВЛЯЕТСЯ (старый путь не ломается). Popup — **дополнительно** для
  preview/быстрого выбора.

## Реализация
Добавить второй source в `autocompletion.override` (или расширить `hybridSource`) — источник для
пресетов `\` и `%`:

1. Новый source-функция (в TtsEditor.vue или useTextCompletion.ts), сигнатура как у CompletionSource:
   ```ts
   // возвращает CompletionResult | null
   function presetSource(context: CompletionContext): CompletionResult | null {
     // триггер: токен \xxx или %xxx перед курсором (до пробела)
     const before = context.matchBefore(/\\[^\s\\]*|%[^\s%]*/)
     if (!before) return null
     const prefix = before.text   // напр. "\na" или "%ad"
     const isUsername = prefix.startsWith('%')
     const keyPart = prefix.slice(1).toLowerCase()
     const map = isUsername ? usr.value : rep.value
     const entries = Object.entries(map)
       .filter(([k]) => k.toLowerCase().startsWith(keyPart))
       .slice(0, 20)   // лимит
     if (entries.length === 0) return null
     return {
       from: before.from,
       options: entries.map(([k, v]) => ({
         label: isUsername ? `%${k}` : `\\${k}`,
         detail: `→ ${v}`,            // preview результата замены
         apply: (view, from, to) => {
           // вставить РЕЗУЛЬТАТ (v) + пробел, переместить курсор
           const insert = v + ' '
           view.dispatch({ changes: { from, to, insert }, selection: { anchor: from + insert.length } })
         },
         // type: типCompletion (текст)
       })),
     }
   }
   ```
2. Зарегистрировать в `autocompletion({ override: [hybridSource, presetSource], ... })`.
   Проверить сигнатуру `hybridSource` — соответствовать ей (CompletionSource).
3. `apply` вставляет **результат** (`v`), не токен — это быстрый путь (как пробел, но из popup).
   Альтернатива: оставить apply = вставка токена (тогда пробел потом подставит). Решить в коде —
   **рекомендация: вставлять результат** (v + ' '),consistent с поведением пробела, popup =
   preview + мгновенная вставка.
4. Trigger-символы: добавить `\` и `%` в `explicitTrigger`/`triggerCharacters` если autocompletion
   это поддерживает (проверить API @codemirror/autocomplete — `autocompletion` опции). Иначе
   `matchBefore` сработает по мере ввода.

## Edge cases
- Пустые rep/usr → source возвращает null (popup не мешает).
- `\` без совпадений по префиксу → null (не показывать пустой popup / шум).
- Не ломать пробел-подстановку (createKeymap handler пробела — не трогать).
- `closeOnBlur`, `selectOnOpen` — оставить как в текущем autocompletion.

## Верификация (DeepSeek прогоняет)
- `npx vue-tsc --noEmit` → 0 ошибок.
- Проверить: `CompletionResult`/`CompletionContext`/`Completion` типы импортированы из
  `@codemirror/autocomplete` (уже импортируются — `autocompletion` оттуда же).
- Проверить сигнатуру `hybridSource` (useTextCompletion.ts) — presetSource должен совпадать по типу.

## Поведение (runtime)
- Ввести `\na` → popup с «\name → Александр», выбор → вставляется «Александр » (результат + пробел).
- Ввести `%ad` → popup «%admin → Администратор».
- Пробел по-прежнему подставляет (старый путь).
- Пустой список replacements → popup не появляется.
