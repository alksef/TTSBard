# Task 103-round1-01: Save soundpanel position on application exit

Прочитай план `docs/deepseek/plan/103-save-soundpanel-position-on-exit.md` и реализуй его.

Добавь в `src-tauri/src/soundpanel_window.rs` общий безопасный helper сохранения `outer_position()` окна `soundpanel` через `WindowsManager`; используй его внутри существующего `hide_soundpanel_window`. В `src-tauri/src/lib.rs` вызови helper из уже существующего обработчика `tauri::RunEvent::ExitRequested`, рядом с сохранением позиции `playback-control`.

Не меняй `quit_app` и tray-логику, не скрывай окна в lifecycle-хуке, не трогай несвязанные файлы. После изменений выполни `cargo check --manifest-path src-tauri/Cargo.toml` и релевантные тесты.
