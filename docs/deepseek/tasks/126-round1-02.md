# Task 126-round1-02: UI локальной клавиши главного окна

Реализуй UI-часть по плану `docs/deepseek/plan/126-2026-07-17-main-window-return-focus-hotkey.md`.
Backend-команда и DTO для `return_previous_window` уже должны быть добавлены задачей
`126-round1-01`.

## Область файлов

- `src/types/settings.ts`
- `src/components/HotkeysPanel.vue`
- компонент/корневой main-window слой, где безопасно устанавливаются локальные keydown
  listeners (найди фактический lifecycle главного окна)

## Требования

1. Добавь `return_previous_window: HotkeyDto` в TypeScript DTO.
2. В HotkeysPanel добавь строку в отдельном блоке «Главное окно»: label «Вернуть фокус
   в предыдущее окно», отображение, запись, отмена и reset. Дефолт backend должен
   отображаться как `Ctrl+F`.
3. Запись использует физический `KeyboardEvent.code` для основной клавиши, но сохраняет
   формат, совместимый с backend parser; не ломай существующие global rows.
4. Добавь локальный listener только для main window. Совпадение проверяет modifiers +
   code, вызывает `invoke('return_to_previous_window')`, делает preventDefault только
   при совпадении. Listener должен корректно сниматься при unmount.
5. Не добавляй listener в playback-control/soundpanel и не меняй NumPad Control.
6. Сохрани существующие CSS-паттерны HotkeysPanel; icon-only кнопки должны иметь title
   и aria-label.

После изменений запусти `npx vue-tsc --noEmit` и `git diff --check`.
