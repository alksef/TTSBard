# План: настраиваемое скрытие звуковой панели при потере фокуса

## Цель

Сохранить обычное поведение звуковой панели: после показа клик вне неё скрывает панель. При этом добавить возможность временно отключить автоскрытие, чтобы спокойно перетащить и настроить её положение.

## Решение

Добавить в `SoundPanelWindowSettings` поле `hide_on_blur` с default `true`. В общих настройках интерфейса добавить checkbox «Скрывать при потере фокуса». Обработчик `Focused(false)` читает это значение из `WindowsManager`: при `true` скрывает панель, при `false` оставляет открытой. Значение click-through остаётся отдельной настройкой.

## Область изменений

- Rust: `src-tauri/src/config/windows.rs`, `src-tauri/src/soundpanel/bindings.rs`, `src-tauri/src/soundpanel/mod.rs`, `src-tauri/src/lib.rs`
- Frontend: `src/components/settings/SettingsInterface.vue`, `src/types/settings.ts`

## Приёмка

1. Старые `windows.json` без нового поля загружаются с `hide_on_blur = true`.
2. При включённой настройке клик вне панели скрывает её.
3. При выключенной настройке потеря фокуса не скрывает панель, поэтому её можно перемещать.
4. Настройка сохраняется и синхронизируется через общий settings cache.
5. Click-through и stay-visible для воспроизведения не смешиваются с новой настройкой.
