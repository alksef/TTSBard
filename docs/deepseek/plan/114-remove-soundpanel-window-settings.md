# План: убрать дублирующие настройки окна из SoundPanelTab

## Проблема

В `SoundPanelTab.vue` остался старый блок настроек floating-окна: цвет, прозрачность, click-through и режим оставления окна видимым. Настройки оформления уже вынесены в раздел общих настроек интерфейса, поэтому в разделе звуковой панели UI дублируется.

## Решение

Удалить из `SoundPanelTab.vue` весь блок «Внешний вид плавающего окна». Удалить из `useSoundPanel.ts` только ставшие невостребованными UI-состояния, preview и функции сохранения/загрузки этих полей. Backend-команды и модель настроек не удалять: floating-окно и раздел общих настроек продолжают их использовать.

## Область изменений

- `src/components/SoundPanelTab.vue`
- `src/composables/useSoundPanel.ts`

## Приёмка

1. В разделе звуковой панели отсутствует блок настроек окна, включая click-through и «оставлять окно видимым».
2. Остальные функции панели — наборы, привязки, добавление/удаление и тест звука — работают без изменений.
3. В `useSoundPanel` не остаётся неиспользуемых UI-состояний и обработчиков, относящихся только к удалённому блоку.
4. Backend-команды/настройки окна не удаляются.
5. Проходят `npx vue-tsc --noEmit`, `cargo check` и `git diff --check`.
