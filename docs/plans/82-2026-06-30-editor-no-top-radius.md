# Plan 82: Убрать верхнее скругление поля под табами (визуальное)

**Дата:** 2026-06-30
**Статус:** ✅ частично выполнено (правка внесена напрямую — тривиально), план для фиксации
**Связано:** план 77 (editor tabs), stage 06.

## Контекст / проблема
После плана 77 (табы) строка табов находится **над** текстовым полем (CodeMirror). Табы имеют
скругление только сверху (`border-radius: 6px 6px 0 0`), а текстовое поле `.cm-editor` имело
скругление **со всех сторон** (`borderRadius: '18px'`). Визуально верхние скруглённые углы поля
не сочетаются с прямыми углами табов сверху — выглядит «оторванным».

## Решение
В теме CodeMirror `ttsTheme` (`src/components/editor/TtsEditor.vue:53-61`, правило `'&'` =
`.cm-editor`) убрать верхнее скругление:
```ts
'&': {
  border: '1px solid var(--color-border-strong)',
  // Верхние углы без скругления — сочетается со строкой табов сверху.
  borderRadius: '0 0 18px 18px',   // было '18px'
  ...
}
```

## Статус
- ✅ Правка внесена напрямую (тривиально, одна строка в теме).
- ⏳ Требует визуальной проверки в собранном приложении (сборка сейчас блокирована нехваткой
  места на диске D:, см. отдельную заметку).

## Критерии готовности
- Верхние углы текстового поля прямые, нижние — скруглённые (`0 0 18px 18px`).
- Поле визуально примыкает к строке табов как единый блок.
- В minimal-mode (без табов) — допустимо: нижние углы скруглены, верхние прямые (не критично,
  поле и так на весь min-height). Если в minimal выглядит плохо — добавить scoped-override в
  InputPanel `.minimal-wrapper :deep(.cm-editor) { border-radius: 18px }` (будущий этап, если
  потребуется).
