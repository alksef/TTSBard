## UI Доработки (стили)

### Исправленные проблемы

1. **Padding конфликты** — `.openai-save-button` с `!important`
2. **Margin-bottom** — 8px для кнопки сохранения
3. **Transition** — изменён с `background` на `filter 0.2s`
4. **Label alignment** — `min-width: 60px` для вертикального выравнивания
5. **Pattern унификация** — Z.ai использует button-row pattern

## TypeScript Типы (Phase 1)

```typescript
// AiProviderType
export const AiProviderType = {
  OpenAi: 'openai',
  ZAi: 'zai'  // Z.ai (capital Z)
} as const

export type AiProviderType = (typeof AiProviderType)[keyof typeof AiProviderType]

// AiOpenAiSettingsDto
export interface AiOpenAiSettingsDto {
  api_key?: string
  use_proxy?: boolean
}

// AiZAiSettingsDto
export interface AiZAiSettingsDto {
  url?: string
  token?: string
}

// AiSettingsDto
export interface AiSettingsDto {
  provider: AiProviderType
  openai: AiOpenAiSettingsDto
  zai: AiZAiSettingsDto
  prompt: string
}
```
