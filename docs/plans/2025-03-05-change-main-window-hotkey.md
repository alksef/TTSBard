# План: Изменение горячей клавиши вызова главного окна

## Задача
Изменить горячую клавишу вызова главного окна с `Ctrl+Alt+T` на `Ctrl+Shift+F3`.
Реализовать вывод окна поверх всех окон только для фокуса (временно).
Добавить разворачивание окна если оно скрыто в трей.
Игнорировать хоткей если окно уже в фокусе.

## Ответы на вопросы

1. **Как работает "поверх всех окон"?**
   - Только для фокуса - окно становится always-on-top, получает фокус, затем возвращается в нормальное состояние

2. **Что делать если окно скрыто в трее?**
   - Развернуть из минимизированного/скрытого состояния

3. **При повторном нажатии хоткея:**
   - Игнорировать если окно уже в фокусе

## Изменения

### 1. Обновление `src-tauri/src/hotkeys.rs`

**Изменить хоткей:**
- Удалить: `Ctrl+Alt+T` (Shortcut с `Code::KeyT`)
- Добавить: `Ctrl+Shift+F3` (Shortcut с `Code::F3`)

**Обновить обработчик:**
```rust
// Было:
let ctrl_alt_t = Shortcut::new(Some(Modifiers::CONTROL | Modifiers::ALT), Code::KeyT);

// Станет:
let ctrl_shift_f3 = Shortcut::new(Some(Modifiers::CONTROL | Modifiers::SHIFT), Code::F3);
```

**Обновить логику обработчика:**
1. Проверить - если окно уже в фокусе, игнорировать
2. Проверить - если окно скрыто (hidden), показать его
3. Установить `set_always_on_top(true)`
4. Вызвать `set_focus()`
5. После получения фокуса убрать `always_on_top`

### 2. Обновить `unregister_hotkeys`
- Заменить удаление `ctrl_alt_t` на `ctrl_shift_f3`

### 3. Обновить `src-tauri/src/lib.rs`

**Добавить обработчик события фокуса главного окна:**
```rust
tauri::WindowEvent::Focused(focused) => {
    if !*focused && window.label() == "main" {
        // Окно потеряло фокус - убрать always-on-top
        let _ = window.set_always_on_top(false);
    }
}
```

### 4. Обновить `src/components/InfoPanel.vue`

**Изменить документацию:**
```html
<!-- Было: -->

<span>Показать главное окно</span>

<!-- Станет: -->
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F3</kbd>
<span>Показать главное окно (поверх всех, для фокуса)</span>
```

### 5. Обновить комментарии в коде

- Обновить комментарии в `hotkeys.rs`
- Обновить документацию к функции `initialize_hotkeys`

## Примечания

- В Tauri методы для работы с окнами:
  - `window.is_visible()` - проверка видимости
  - `window.show()` - показать окно
  - `window.hide()` - скрыть окно
  - `window.set_focus()` - сфокусировать окно
  - `window.is_focused()` - проверка фокуса
  - `window.set_always_on_top(bool)` - установить поверх всех
  - `window.unminimize()` - развернуть из минимизированного состояния
