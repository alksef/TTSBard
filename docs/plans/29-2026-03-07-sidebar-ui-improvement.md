# План: Улучшение UI сайдбара

**Дата:** 2026-03-07
**Задача:** Редизайн сайдбара с группировкой, новыми иконками и возможностью сворачивания

**Статус:** ✅ ЗАВЕРШЕНО

## Требования

Из `docs/ideas/ui/promt.txt`:

1. **Группировка по смыслу** - разделение функций на логические блоки
2. **Упрощение визуального стиля** - убрать рамки, единый стиль иконок
3. **Акцент на активном пункте** - вертикальная полоска для активного элемента
4. **Оптимизация пространства** - сворачивание панели

## Выбранный подход

- **Вариант 3** - внешняя библиотека иконок (lucide-vue-next)
- **Цвета** - сохранить существующие (синий акцент как кнопка "Озвучить")
- **Сохранение состояния** - сворачивание сохраняется между перезапусками
