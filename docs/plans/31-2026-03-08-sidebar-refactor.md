# План: Рефакторинг сайдбара

**Дата:** 2026-03-08
**Задача:** Улучшение UX сайдбара - убрать лишние элементы, добавить плавающую кнопку сворачивания

## Требования

1. **Убрать заголовок "TTSBard"** из header сайдбара
2. **Плавающая кнопка сворачивания**:
   - Позиция: по центру вертикали на правой границе сайдбара
   - Форма: круглая
   - При сворачивании: остаётся на месте (не сдвигается)
   - Z-index: поверх всего контента
   - Видимость: всегда видна
   - Эффекты: подсветка при hover
3. **Убрать заголовки групп** ("ГЛАВНОЕ", "Инструменты", "ИНТЕГРАЦИЯ")
4. **Оставить разделители** между группами кнопок
5. **Изменить ширину сайдбара**:
   - Обычный: 248px → 200px
   - Свёрнутый: 82px → 70px
6. **Версию в футере**: оставить как есть
7. **Мобильная версия**: игнорировать (десктопное приложение)

## План реализации

### Шаг 1: Sidebar.vue - удаление заголовка TTSBard
- [ ] Удалить `<div class="brand-copy">` и `<div class="brand-title">` из template
- [ ] Удалить `.brand-copy` и `.brand-title` из `<style scoped>`
- [ ] Упростить `.sidebar-header` (убрать flex layout для brand)

### Шаг 2: Sidebar.vue - удаление заголовков групп
- [ ] Убрать `title` из `SidebarGroup` интерфейса (сделать опциональным или удалить)
- [ ] Удалить `<div class="sidebar-group-title">` из template
- [ ] Удалить `.sidebar-group-title` из `<style scoped>`
- [ ] Оставить `.sidebar-divider` между группами

### Шаг 3: Sidebar.vue - изменение ширины сайдбара
- [ ] Изменить `.sidebar` ширину: 248px → 200px
- [ ] Изменить `.sidebar-collapsed` ширину: 82px → 70px
- [ ] Обновить `.sidebar-header` padding для новой ширины

### Шаг 4: Sidebar.vue - плавающая кнопка сворачивания
- [ ] Убрать кнопку из `.sidebar-header`
- [ ] Создать новый компонент кнопки с `position: absolute`
- [ ] Позиция: `right: -17px`, `top: 50%`, `transform: translateY(-50%)`
- [ ] Добавить `z-index: 1000` (поверх всего контента)
- [ ] Стиль: круглая, тень, градиентная граница
- [ ] Hover: подсветка, лёгкое масштабирование
- [ ] Иконка: оставить ChevronLeft/ChevronRight

### Шаг 5: CSS стили для плавающей кнопки
- [ ] `.collapse-toggle-floating` - базовый стиль
- [ ] `box-shadow` для отделения от фона
- [ ] `:hover` с ярким свечением
- [ ] Плавные transition анимации
- [ ] Адаптация для свёрнутого состояния

## Примечания

1. Плавающая кнопка должна быть визуально заметной, но не навязчивой
2. При сворачивании кнопка остаётся на том же месте по горизонтали
3. Разделители между группами сохраняют структуру без заголовков
4. Ширина 200px оптимальна для чтения текста кнопок
