# DECISION-017 — Лицензия GPL-3.0-only

**Статус:** `accepted`
**Связано:** [ROADMAP-039](../roadmap/completed/039-piper-runtime-feasibility.md)

## Контекст

Встроенный Piper runtime статически включает eSpeak NG через `espeak-rs-sys`.
Исходники eSpeak NG распространяются по GPLv3, хотя metadata wrapper-crate
указывает MIT и поэтому не отражает условия включённого native-кода.

## Решение

Оригинальный код TTSBard начиная с этого решения распространяется по
`GPL-3.0-only`. Полный неизменённый текст лицензии находится в корневом
`LICENSE`, SPDX identifier указан в Cargo/npm manifests и Tauri bundle.

Сторонние библиотеки, словари, модели и vendored-код не перелицензируются: для
них должны сохраняться исходные copyright notices и собственные лицензии.
Лицензия голосовой модели проверяется отдельно до её распространения вместе с
приложением.

## Последствия

- Уже полученные копии и ранее опубликованные версии сохраняют лицензионные
  условия, с которыми они распространялись; новое решение применяется к
  текущей и последующим версиям оригинального кода TTSBard.
- При распространении бинарной сборки должен быть доступен соответствующий
  исходный код и текст GPLv3.
- Производные версии TTSBard распространяются на совместимых условиях GPLv3.
- Tauri включает корневой `LICENSE` в поддерживаемые bundles через
  `bundle.licenseFile`.
- Новая dependency или bundled resource проверяется не только по manifest
  metadata, но и по фактически включённому native/model содержимому.
- Полный third-party notice и проверка release bundle ведутся отдельно в
  [TASK-119](../tasks/119-third-party-license-notices.md).
