# DECISION-012 — CSS tokens и темы

**Статус:** `accepted`

## Контекст

Несколько панелей и окон должны сохранять единый вид в тёмной и светлой теме.

## Решение

Цвета, поверхности, состояния и размеры задаются семантическими CSS variables.
Светлая тема переопределяет tokens через theme attribute.

## Последствия

Компоненты не вводят локальную палитру без необходимости. Новое состояние
проверяется в обеих темах; существующий компонент служит визуальным эталоном.
