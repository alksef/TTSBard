# DECISION-015 — Отдельный F6-режим

**Статус:** `superseded`
**Заменено:** [DECISION-009](./009-windows-and-hotkeys.md)

## Контекст

Ранний интерфейс использовал F6 как отдельный режим, меняющий поведение Enter и
закрытие floating window.

## Решение

Не сохранять сущность «F6 mode». Поведение задаётся настройками режима, а
сочетания клавиш — configurable hotkeys.

## Последствия

Документация и код не должны предполагать фиксированную клавишу F6.
