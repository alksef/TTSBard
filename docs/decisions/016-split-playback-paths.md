# DECISION-016 — Раздельные playback-пути

**Статус:** `superseded`
**Заменено:** [DECISION-006](./006-pcm-audio-pipeline.md)

## Контекст

Ранее TTS предлагалось отправлять системному плееру, а SoundPanel проигрывать
внутри процесса через rodio.

## Решение

Отказаться от разных архитектурных путей. TTS и replay используют управляемый
PCM pipeline и `PlaybackManager`.

## Последствия

Очередь, эффекты и dual output имеют единое место управления.
