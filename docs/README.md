# TTSBard - Руководство пользователя

> **TTS-приложение для общения с минимальным отвлечением от работы**
> Версия: 0.3.0-dev
> Платформа: Windows 10/11
> Дата обновления: 2026-04-15

---

## Краткое описание

**TTSBard** — это приложение для синтеза речи (Text-to-Speech), которое работает в фоновом режиме и доступно мгновенно по горячей клавише в любом приложении. Поддерживает 4 TTS-провайдера: OpenAI TTS, Silero Bot (Telegram), Fish Audio и Local TTS.

---

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl + Shift + F1` | Режим перехвата текста / Показать плавающее окно |
| `Ctrl + Shift + F2` | Показать звуковую панель |
| `Ctrl + Shift + F3` | Показать главное окно (поверх всех, для фокуса) |

### В режиме перехвата:

| Клавиша | Действие |
|---------|----------|
| `F8` | Переключить раскладку клавиатуры (EN ↔ RU) |
| `F6` | Переключить режим: Enter не закрывает окно / Enter закрывает окно |
| `Enter` | Отправить текст в TTS (закрыть окно, если режим F6 выключен) |
| `Escape` | Очистить текст, отключить перехват, скрыть окно |
| `Backspace` | Удалить последний символ |
| `Space` | Добавить пробел (с автозаменой пресетов) |

**Режим F6:** Когда включён, Enter отправляет текст в TTS, но окно остаётся открытым для продолжения ввода. Удобно для последовательных сообщений.

## Настройка горячих клавиш

Приложение поддерживает кастомизацию горячих клавиш для всех основных функций:

1. Откройте панель **"Горячие клавиши"** в боковом меню
2. Выберите функцию для настройки
3. Нажмите кнопку записи и нажмите нужную комбинацию клавиш
4. Сохраните изменения

**Доступные для настройки функции:**
- Режим перехвата текста (по умолчанию: `Ctrl+Shift+F1`)
- Звуковая панель (по умолчанию: `Ctrl+Shift+F2`)
- Главное окно (по умолчанию: `Ctrl+Shift+F3`)

## Структура приложения

### Боковая панель (Sidebar)

Главное окно содержит боковую панель с навигацией по разделам:

**ГЛАВНОЕ:**
- **Текст** — основной редактор для ввода и озвучивания текста
- **Руководство** — справочная информация и горячие клавиши
- **TTS** — настройка TTS-провайдеров и голосов
- **Аудио** — настройка устройств вывода и громкости

**Инструменты:**
- **Звуковая панель** — назначение звуков на клавиши A-Z

**Дополнительно:**
- **Быстрая вставка** — настройка автозамены и пресетов
- **WebView** — настройка сервера для OBS
- **Twitch** — интеграция с Twitch чатом
- **Настройки** — общие настройки приложения
- **Горячие клавиши** — настройка комбинаций клавиш

### Плавающее окно

Появляется при включении режима перехвата (горячая клавиша `Ctrl+Shift+F1`).

**Индикатор раскладки:**
- 🟢 **EN** — английская (зелёный)
- 🟠 **RU** — русская (оранжевый)

## TTS провайдеры

Приложение поддерживает 4 TTS-провайдера:

### OpenAI TTS
- 6 естественных голосов: alloy, echo, fable, onyx, nova, shimmer
- Настройка скорости (0.25-4.0)
- Поддержка прокси
- Требуется API ключ

### Silero Bot (Telegram)
- Интеграция с Telegram ботом @SileroBot
- Авторизация через телефон
- Работает без API ключа
- Выбор голоса через бота

### Fish Audio
- Высококачественный синтез речи
- Настройка формата аудио (mp3, wav, ogg)
- Регулировка temperature и sample rate
- Поддержка прокси
- Требуется API ключ

### Local TTS
- Режим "Locally Hosted" из TTS Voice Wizard
- HTTP запросы к локальному серверу (TITTS.py или совместимый)
- Работает без интернета
- URL по умолчанию: `http://127.0.0.1:8124`

## Обработка текста

Автоматическая обработка текста перед отправкой в TTS выполняется в три этапа:

1. **Управление отправкой** — префиксы `!` и `!!` контролируют маршрутизацию
2. **Быстрая вставка** — пресеты `\ключ` и `%юзернейм` для замены текста
3. **Конвертация чисел** — числа преобразуются в текст с согласованием рода

### Управление отправкой

Используйте префиксы в начале текста для контроля отправки в Twitch и WebView:

| Префикс | Описание | Пример | Twitch | WebView | TTS |
|---------|----------|--------|--------|---------|-----|
| Без префикса | Отправить везде | `Привет всем` | ✅ | ✅ | ✅ |
| `!` | Только WebView | `!Привет` | ❌ | ✅ | ✅ |
| `!!` | Только TTS | `!!Привет` | ❌ | ❌ | ✅ |

**Примеры использования:**
```
!Только для OBS — озвучить, но не в Twitch
!!Локальное сообщение — только для TTS
Обычный текст — отправить везде
```

### Быстрая вставка

| Синтаксис | Описание | Пример | Результат |
|-----------|----------|--------|----------|
| `\ключ` | Замена текста по коду | `\name` + пробел | Александр |
| `%юзернейм` | Замена имени пользователя | `%admin` + пробел | Администратор |

**Настройка:** Панель "Быстрая вставка" → Списки замен / Список юзернеймов

### Конвертация чисел

Числа автоматически преобразуются в текст с согласованием рода следующего слова:

| Ввод | Результат |
|------|-----------|
| `У меня 5 яблок` | `У меня пять яблок` |
| `1 книга` | `одна книга` |
| `2 книги` | `две книги` |
| `5 книг` | `пять книг` |
| `123` | `сто двадцать три` |
| `-10` | `минус десять` |

**Правила:**
- Числа `1` и `2-4` согласуются с родом следующего слова
- Числа `5+` используют множественное число

## AI коррекция текста

Приложение поддерживает автоматическую и ручную коррекцию текста с помощью AI.

### Режимы редактора

**Quick Editor (Быстрый редактор):**
- Enter/Esc сворачивают окно после отправки
- Текст очищается автоматически
- Идеально для быстрого ввода и озвучивания

**AI Editor (AI редактор):**
- Автоматическая коррекция текста перед отправкой
- Поддержка OpenAI и Z.ai провайдеров
- Ручная коррекция по кнопке с иконкой ✨

### Настройка AI коррекции

1. Откройте панель **Настройки**
2. Перейдите в раздел **AI**
3. Выберите провайдер (OpenAI или Z.ai)
4. Введите API ключ
5. Включите опцию **"AI Editor"** в разделе **Editor**
6. Сохраните настройки

**Примечание:** Для работы AI коррекции требуется настроенный API ключ выбранного провайдера.

## Звуковая панель (Саунпад)

Звуковая панель позволяет мгновенно воспроизводить звуковые эффекты по нажатию клавиши.

### Использование

1. Откройте панель **Звуковая панель** в боковом меню
2. Назначьте аудиофайл на букву (A-Z)
3. Нажмите `Ctrl+Shift+F2` — появится панель с клавиатурой
4. Нажмите любую букву — звук воспроизводится мгновенно

### Возможности

- До 26 одновременных привязок (клавиши A-Z)
- Форматы: MP3, WAV, OGG, FLAC
- Глобальные горячие клавиши для быстрого доступа
- Настройка громкости для каждого звука

### Горячие клавиши

- `Ctrl+Shift+F2` — показать/скрыть панель
- `A-Z` — воспроизвести назначенный звук (работает глобально)

## Интеграции

### WebView сервер (OBS)

Отображение текста в реальном времени в OBS Studio или браузере через Server-Sent Events (SSE).

**Настройка:**
1. Включите WebView сервер в настройках (по умолчанию порт 10100)
2. Добавьте Browser Source в OBS: `http://localhost:10100`
3. Отправьте текст в TTS — текст появится в OBS мгновенно

**Возможности:**
- SSE (Server-Sent Events) для real-time обновлений
- Настраиваемый порт и bind address (0.0.0.0 / 127.0.0.1)
- Полная кастомизация HTML/CSS шаблонов
- Токен-аутентификация для внешних подключений
- UPnP — автоматический проброс портов на роутере

### Twitch Chat

Отправка TTS сообщений в Twitch чат для дублирования.

**Настройка:**
1. Создайте Twitch бота на [twitchtokengenerator.com](https://twitchtokengenerator.com)
2. Скопируйте OAuth токен со страницы (только сам токен, без префикса)
3. Введите данные в настройках Twitch Chat
4. Нажмите Start для подключения к чату

**Поля настроек:**
- **Username** — имя бота (без @)
- **Token** — только токен (без `oauth:`)
- **Channel** — канал (имя или URL)

## Первая настройка

### Выбор TTS провайдера

При первом запуске выберите один из 4 доступных TTS-провайдеров:

**OpenAI TTS:**
1. Получите API ключ на https://platform.openai.com/api-keys
2. Скопируйте ключ (начинается с `sk-`)
3. Откройте панель **TTS** → вкладка **OpenAI**
4. Вставьте ключ и нажмите **"Сохранить настройки"**

**Silero Bot:**
1. Откройте панель **TTS** → вкладка **Silero**
2. Нажмите **"Подключить Telegram"**
3. Следуйте инструкциям в открывшемся окне
4. Выберите голос через бота @SileroBot

**Fish Audio:**
1. Получите API ключ на сайте Fish Audio
2. Откройте панель **TTS** → вкладка **Fish Audio**
3. Вставьте ключ и настройте параметры
4. Нажмите **"Сохранить настройки"**

**Local TTS:**
1. Убедитесь, что локальный TTS сервер запущен (TITTS.py)
2. Откройте панель **TTS** → вкладка **Local**
3. Укажите URL сервера (по умолчанию `http://127.0.0.1:8124`)
4. Нажмите **"Сохранить"**

## Использование

### Вариант 1: Режим перехвата (рекомендуется)

1. Нажмите `Ctrl + Shift + F1`
2. Появится плавающее окно с индикатором EN
3. Набирайте текст (клавиатура перехватывается globally)
4. Нажмите `Enter` для озвучивания

### Вариант 2: Главное окно

1. Нажмите `Ctrl + Shift + F3` для показа главного окна
2. Введите текст в поле редактора
3. Нажмите кнопку **"Озвучить"** или `Enter`

### Вариант 3: Быстрый редактор

1. Включите **Quick Editor** в настройках (Настройки → Editor)
2. Нажмите `Ctrl + Shift + F3` для показа главного окна
3. Введите текст и нажмите `Enter`
4. Окно автоматически скроется после отправки

## Системные требования

### Минимальные требования

- **ОС:** Windows 10/11 (64-bit)
- **Процессор:** Intel Core i3 или эквивалент
- **Память:** 4 GB RAM
- **Место на диске:** 100 MB
- **Интернет:** Для облачных TTS-провайдеров (OpenAI, Fish Audio, Silero)

### Рекомендуемые требования

- **ОС:** Windows 11 (64-bit)
- **Процессор:** Intel Core i5 или эквивалент
- **Память:** 8 GB RAM
- **Место на диске:** 500 MB
- **Интернет:** Стабильное подключение

### Примечания

- Для **Local TTS** интернет не требуется
- Для работы с правами Administrator могут потребоваться дополнительные настройки
- Приложение не перехватывает клавиши от приложений с правами Administrator
- Не может блокировать `Ctrl+Alt+Del` (Secure Desktop)

## Возможные проблемы

### Горячие клавиши не работают

**Причина:** Хоткей может быть занят другим приложением

**Решение:**
1. Закройте приложения, которые могут использовать эти клавиши
2. Или настройте другие комбинации в панели **"Горячие клавиши"**

### TTS не озвучивает текст

**Проверьте:**
1. Выбран ли TTS-провайдер в панели **TTS**
2. Введён ли API ключ (для OpenAI, Fish Audio)
3. Подключён ли Silero Bot (для Silero)
4. Запущен ли локальный сервер (для Local TTS)

### Плавающее окно не перехватывает клавиши

**Причина:** Окно получило фокус после перетаскики или взаимодействия

**Решение:**
1. Скройте окно (`Escape`)
2. Снова активируйте режим перехвата (`Ctrl+Shift+F1`)
3. Не взаимодействуйте с окном после появления
