# TTS Application - User Guide

> **Приложение для работы с TTS (Text-to-Speech)**
> Версия: 0.1.0
> Платформа: Windows

---

## Краткое описание

Приложение перехватывает ввод с клавиатуры по горячей клавише и отправляет текст в OpenAI TTS для озвучивания.

---

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl + Alt + Shift + C` | Включить режим перехвата (показать плавающее окно) |
| `Ctrl + Alt + T` | Показать главное окно |
| `F8` | Переключить раскладку EN/RU (в режиме перехвата) |

### В режиме перехвата:

| Клавиша | Действие |
|---------|----------|
| `Enter` | Отправить текст в TTS и закрыть окно |
| `Escape` | Отменить ввод и закрыть окно |
| `Backspace` | Удалить последний символ |
| `Печатаемые` | Сбор текста в буфер |

---

## Автозамена текста

### Управление отправкой

Используйте префиксы в начале текста для управления отправкой в Twitch и WebView:

| Префикс | Описание | Пример | Twitch | WebView | TTS |
|---------|----------|--------|--------|---------|-----|
| Без префикса | Отправить везде | `Привет всем` | ✅ | ✅ | ✅ |
| `!` | Только WebView | `!Привет` | ❌ | ✅ | ✅ |
| `!!` | Только TTS | `!!Привет` | ❌ | ❌ | ✅ |

**Примеры использования:**
```
!Только для OBS — озвучить, но не в Twitch
!!Локальное сообщение — только для TTS
Обычный текст — отправить везде
```

### Быстрая вставка

- **Список замен** — используйте `\ключ` для замены фраз (настраивается в панели "Автозамена")
- **Список юзернеймов** — используйте `%юзернейм` для замены имён пользователей

### Конвертация чисел

Числа автоматически преобразуются в текст с согласованием рода следующего слова:

| Ввод | Результат |
|------|-----------|
| `У меня 5 яблок` | `У меня пять яблок` |
| `1 книга` | `одна книга` |
| `2 книги` | `две книги` |
| `5 книг` | `пять книг` |
| `123` | `сто двадцать три` |
| `-10` | `минус десять` |

**Правила:**
- Числа `1` и `2-4` согласуются с родом следующего слова
- Числа `5+` используют множественное число

## Структура приложения

### Главное окно

**Меню:**
- **Ввод** — форма для ручного ввода текста
- **Настройки** — настройки приложения

**Ввод:**
1. Введите текст в поле
2. Нажмите "Озвучить"

**Настройки:**
- ☐ **Включить режим перехвата** — переключение hotkey режима
- **API ключ OpenAI** — сохраните ваш ключ

---

### Плавающее окно

Появляется при включении режима перехвата.

**Индикатор раскладки:**
- 🟢 **EN** — английская (зелёный)
- 🟠 **RU** — русская (оранжевый)

## Установка и запуск

### Требования

- Windows 10/11
- Node.js 18+
- Rust 1.80+

### Запуск

```bash
cd D:\RustProjects\app-tts-v2
npm run tauri dev
```

### Сборка релиза

```bash
npm run tauri build
```

## Первая настройка

1. Запустите приложение
2. Откройте **Настройки**
3. Введите ваш **OpenAI API ключ**
4. Нажмите **Сохранить**

---

## Использование

### Вариант 1: Горячие клавиши (быстро)

1. Нажмите `Ctrl + Alt + Shift + C`
2. Появится плавающее окно с индикатором EN
3. Набирайте текст
4. Нажмите `Enter` для озвучивания

### Вариант 2: Ручной ввод

1. Откройте **Ввод** в главном окне
2. Введите текст
3. Нажмите **Озвучить**

---

## Переключение раскладки

В режиме перехвата:
- Нажмите **F8** для переключения EN ↔ RU
- Индикатор в заголовке окна изменит цвет

---

## Возможные проблемы

### Горячие клавиши не работают

**Причина:** Хоткей может быть занят другим приложением

**Решение:**
1. Закройте приложения, которые могут использовать эти клавиши
2. Или используйте переключатель в Настройках

---

## Системные требования

### API ключ OpenAI

Для работы необходим ключ OpenAI API:

1. Получите ключ на https://platform.openai.com/api-keys
2. Скопируйте ключ (начинается с `sk-`)
3. Вставьте в Настройки приложения
