# Дорожная карта TTSBard

Этот раздел описывает путь развития продукта. Подробные roadmap items
распределяются по фактическому состоянию:

- [`active/`](./active/) — исследуемые, запланированные, выполняемые и
  отложенные направления;
- [`completed/`](./completed/) — завершённые направления, для которых полезно
  сохранить итог и оставшийся scope;
- [`rejected/`](./rejected/) — осознанно отклонённые направления с причиной и
  условиями возможного пересмотра.

До завершения этапа 2 действующим источником старых roadmap-материалов остаётся
`docs/stage/`. Новые элементы уже следует создавать здесь.
