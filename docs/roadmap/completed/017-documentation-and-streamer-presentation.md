## 📦 Блоки презентации для стримеров

Презентация ориентирована на стримеров и контент-мейкеров. Она должна содержать 11 ключевых блоков:

### 1. Продвинутый Редактор текста
* **Фишка:** Разработан специально под быстрый и точный набор текста «вслепую» во время игры.
* **Киллер-фичи:**
  - **Офлайн-проверка орфографии (Spellcheck):** Быстрая подсветка опечаток прямо во время ввода без обращения к внешним сервисам.
  - **Умное автодополнение (Autocomplete):** Подсказки слов на основе истории уже набранных фраз в текущей сессии, позволяющие дописывать длинные слова за 1-2 нажатия.
  - **Интеграция с CodeMirror 6:** Быстрый, легковесный интерфейс ввода с поддержкой кастомных горячих клавиш.

### 2. Общий функционал и интерфейс
* **Фишка:** Незаметность и мгновенный доступ.
* **Возможности:**
  - **Глобальные горячие клавиши:** Вызов окна ввода поверх любой игры/приложения в один клик.
  - **Сворачивание в системный трей:** Приложение не загромождает панель задач и работает в фоне.
  - **Компактный режим:** Минималистичный плавающий интерфейс ввода (floating window), который не закрывает экран трансляции.

### 3. Интерактивная Саундпанель (Soundboard)
* **Фишка:** Мгновенный отыгрыш звуковых реакций без потери фокуса.
* **Как работает:**
  - Открывается по сочетанию клавиш.
  - **Активное отслеживание клавиш:** Даже когда панель находится поверх окон, она мгновенно перехватывает нажатия букв (A-Z) для воспроизведения звуков.
  - **Быстрое закрытие:** Панель автоматически закрывается при потере фокуса или нажатии `Esc`, мгновенно возвращая управление в игру.

### 4. Блок быстрой вставки (Препроцессор фраз)
* **Фишка:** Сокращение времени на ввод частых фраз.
* **Возможности:**
  - Шаблоны автозамены (`\ключ` на длинную фразу, например, `\hi` -> `Привет, добро пожаловать на стрим!`).
  - Автоматическая подстановка юзернеймов по коду `%username`.
  - Автозамена срабатывает на лету при нажатии клавиши `Space`.

### 5. Разделение аудиовывода (Dual Audio Output)
* **Фишка:** Чистый звук для стримера и зрителей.
* **Возможности:**
  - Одновременное дублирование синтезированной речи в **наушники стримера** (чтобы слышать, что говорит бот) и в **виртуальный микрофон** (VB-Cable / VoiceMeeter) для отправки голоса в Discord/игру.
  - Раздельная регулировка громкости для каждого направления.

### 6. Интеграция с OBS (WebView сервер)
* **Фишка:** Вывод субтитров и текста на экран трансляции без сторонних виджетов.
* **Возможности:**
  - Встроенный легковесный веб-сервер, который транслирует текст через Server-Sent Events (SSE).
  - Стример просто добавляет как «Источник браузера» (Browser Source) локальный адрес.
  - **Трансляция потока наружу:** Поддержка UPnP для автоматического проброса портов и токен-авторизация позволяют отдавать текстовый поток внешним получателям (например, выводить ваши реплики в OBS другого стримера во время коллаборации).

### 7. Интеграция с Twitch Chat
* **Фишка:** Дублирование реплик в текстовый чат стрима.
* **Возможности:**
  - Встроенный Twitch IRC клиент.
  - Автоматическое подключение при старте.
  - Префиксы маршрутизации: `!` (отправить только в OBS/TTS, но не дублировать в чат Twitch) и `!!` (отправить только локально в TTS).

### 8. Экспериментальный перехват клавиш (Numpad Control)
* **Фишка:** Управление приложением одной рукой на дополнительной клавиатуре.
* **Возможности:**
  - Перевод Numpad-клавиш в режим управления воспроизведением и навигации (например, для людей с серьезными ограничениями моторики или для быстрого контроля на стриме).

### 9. AI-функционал (Коррекция текста)
* **Фишка:** Интеллектуальное исправление «на лету» перед озвучкой.
* **Реализованные возможности:**
  - Быстрая отправка текста в OpenAI/Z.ai для исправления грамматики, пунктуации или изменения стиля фразы по кастомному промпту.
  - Кнопка включения/выключения AI-режима в редакторе.

### 10. Прокси для Telegram-интеграции (Silero Bot)
* **Фишка:** Стабильный доступ к Telegram API при любых сетевых блокировках.
* **Варианты прокси:**
  - **Socks5:** Для работы через локальные программы-туннели (например, V2Ray, Xray).
  - **MTProxy:** Встроенная поддержка прямого подключения к прокси-серверам Telegram без использования стороннего ПО.

### 11. Панель управления воспроизведением (Playback Control)
* **Фишка:** Полный контроль над звучащим голосом в реальном времени.
* **Возможности:**
  - Отдельное плавающее окно управления озвучкой.
  - Кнопка мгновенной остановки воспроизведения (Stop / Panic button), если бот начал читать длинный или некорректный текст.
  - Пауза и навигация по фразам.
