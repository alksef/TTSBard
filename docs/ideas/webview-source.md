# WebView Source - Animated Text Display Server

**Дата:** 2025-03-06
**Статус:** ✅ Утверждён, план реализации готов

---

## Идея

Создать HTTP-сервер который отдаёт веб-страницу с анимированным текстом, отправленным на TTS. Страница может быть добавлена в OBS Studio как Browser Source для отображения текста на стриме.

---

## Ключевые решения (из брейншторма)

| Вопрос | Ответ |
|--------|-------|
| **Цель** | OBS/Стриминг — отображение текста как источника для трансляций |
| **Тип анимации** | Печатная машинка (typing effect) — текст появляется буква за буквой |
| **Накопление текста** | Только текущая фраза — новый текст заменяет предыдущий |
| **Настройки стиля** | Полный контроль HTML/CSS — пользователь может редактировать шаблон |
| **Порт сервера** | Фиксированный (10100 по умолчанию, настраиваемый в UI) |
| **Доступ** | Локальная сеть (0.0.0.0) — можно зайти с других устройств |
| **DNS** | Без DNS — просто IP:порт, кнопка обновления IP в UI |
| **Интеграция** | Через существующую event систему (`AppEvent::TextSentToTts`) |

---

## Техническое решение

### Архитектура

```
TTS Core → Event Channel → WebViewSource Module → WebSocket → OBS Browser Source
```

### Стек технологий

- **HTTP Server:** axum 0.7
- **WebSocket:** tokio-tungstenite 0.21
- **Frontend:** Vue 3 + TypeScript

### Компоненты

1. **HTTP Server** — слушает на `0.0.0.0:10100`, отдаёт HTML страницу
2. **WebSocket Broadcaster** — рассылает текст всем подключённым клиентам
3. **Settings Panel** — Vue компонент для настройки HTML/CSS/порта
4. **Storage** — шаблоны сохраняются в `AppData\ttsbard\webview\`

---

## WebSocket протокол

**Сообщение от сервера:**
```json
{
  "type": "text",
  "text": "Привет, это тест TTS",
  "timestamp": 1709876543000
}
```

**Клиент подключается:** `ws://<IP>:<PORT>/ws`

---

## UI панель настроек

- Включить/выключить сервер
- Порт (10100 по умолчанию)
- Bind address (0.0.0.0 или 127.0.0.1)
- Ссылка для OBS с кнопками копирования и обновления IP
- HTML шаблон (редактируемый textarea)
- CSS стиль (редактируемый textarea)
- Скорость анимации (мс/символ)

---

## Интеграция с TTS

Добавлен новый event `AppEvent::TextSentToTts(String)` который отправляется из всех TTS провайдеров (OpenAI, Silero, Local) перед synthesizing.
