# Context Rules

**ВАЖНО:** При поднятии контекста и исследовании кодовой базы игнорировать следующие папки:

```
docs/plans/     - Планы реализации (исторические)
docs/img*.png   - Скриншоты для документации
```

**Основные файлы документации для контекста:**
- docs/INDEX.md - главный индекс
- docs/cheatsheet.md - быстрые команды
- docs/project-overview.md - обзор проекта
- docs/rust-modules.md - Rust модули
- docs/vue-components.md - Vue компоненты
- docs/architecture.md - архитектура

**Приоритет при исследовании кода:**
1. Исходный код (`src-tauri/src/`, `src/`)
2. Основная документация (выше)
3. Планы и скриншоты - только по явному запросу
